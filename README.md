 ## Overview of Sakhi Finance Simulation App Design
  SakhiFinance App, an interactive financial simulation app where users manage a virtual household budget and make real-life financial decisions.

  The frontend is built using HTML, CSS, and JavaScript, and the backend is built using Node.js and Express.
  The app works in a very simple way. When the user opens the app, she sees big buttons, icons, and voice guidance.

She can select Budget Jars, Decision Simulation, Fraud Alert, or Progress.

The app gives real-life situations and shows immediate consequences of financial decisions.

This helps women gain confidence and practical knowledge.
